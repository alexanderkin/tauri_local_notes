## 6. 你踩过并解决的问题

这个项目里你遇到过很多真实开发问题：

- PowerShell 中 `npm` 被脚本策略拦住
- Rust 环境未安装
- 样式路径不生效
- 保存文件触发 Tauri dev 重启
- 中文乱码导致代码异常
- `window.__TAURI__` 不存在
- `withGlobalTauri` 配置缺失
- 打包时中文 `productName` 导致 MSI 失败
- Windows 图标缓存导致新图标不立刻显示
- JSON 结构升级后旧数据缺少 `id`
- 搜索结果中删除可能删错

这些问题很重要，因为真实项目不会一直顺利。

## 7. 你最应该记住的开发方法

这个项目最重要的学习方法不是某一段代码，而是：

- 一次只做一步
- 每一步都先跑通
- 报错先排错，不急着学新东西
- 先用简单方案，再逐步升级
- 功能完成后及时复盘
- 文档同步记录

这套方法以后做其他项目也能继续用。

## 8. 当前项目的不足

当前应用已经能用，但还不是一个成熟产品。

它还有这些可以继续优化的地方：

- 界面还可以更精致
- 笔记内容不支持换行的复杂显示
- 搜索关键词没有高亮
- 没有分类或标签
- 没有导入导出
- 没有自动备份
- 没有单元测试
- 没有更完整的错误提示

这些不是问题，而是后续升级方向。
