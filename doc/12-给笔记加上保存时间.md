# 第 12 步：给笔记加上保存时间

## 1. 这一步的目标

让每条新保存的笔记都带上保存时间。

这样以后回看笔记时，你就能知道：

- 这条笔记是什么时候记下来的

## 2. 你刚刚做了什么

你修改了：

- `tauri-notes/src-tauri/Cargo.toml`
- `tauri-notes/src-tauri/src/lib.rs`

你完成的事情是：

- 给 Rust 项目增加了时间处理依赖
- 在保存笔记时，顺手把当前时间一起写进文件

## 3. 这一步里最重要的内容

### 在 `Cargo.toml` 里增加依赖

```toml
chrono = "0.4"
```

它的作用是：
让 Rust 可以方便地获取和格式化当前时间。

### 在 Rust 里获取当前时间

```rust
let now = Local::now().format("%Y-%m-%d %H:%M:%S").to_string();
```

你现在可以先这样理解：

- `Local::now()`：获取当前本地时间
- `.format(...)`：把时间整理成好读的格式

## 4. 为什么这一步重要

因为笔记不只是“写了什么”，很多时候还需要知道“什么时候写的”。

这会让你的笔记数据更完整，也更接近真实应用。

## 5. 这一步完成后你学到了什么

- 知道 Rust 项目可以通过 `Cargo.toml` 引入依赖
- 知道可以使用现成工具处理时间
- 知道保存数据时可以顺手补充更多上下文信息

## 6. 这一步的小复盘

你刚刚做的事情是：
让每条新笔记都带上保存时间。

它在整个项目里的作用是：
让笔记内容更完整，也更有实际使用价值。

这一步最需要记住的是：

- `Cargo.toml` 管理 Rust 依赖
- `chrono` 用来处理时间
- 新功能不一定总是大功能，小增强也很重要
