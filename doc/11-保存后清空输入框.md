# 第 11 步：保存后自动清空输入框

## 1. 这一步的目标

让用户在保存成功后，输入框自动清空，并把光标放回标题输入框。

这样连续录入多条笔记时会更顺手。

## 2. 你刚刚做了什么

你修改了：

- `tauri-notes/src/main.js`

你完成的事情是：

- 保存成功后清空标题输入框
- 保存成功后清空内容输入框
- 光标自动回到标题输入框

## 3. 这一步里最关键的代码

```javascript
titleInput.value = "";
contentInput.value = "";
titleInput.focus();
```

可以先这样理解：

- `.value = ""`：把输入框内容清空
- `.focus()`：把光标主动放到某个输入框里

## 4. 为什么这一步重要

这一步没有增加“新大功能”，但增加了很重要的“使用体验”。

如果每次保存后还要手动删旧内容，再点回标题框，连续记录会很麻烦。

所以这一步的意义是：
让应用更像一个真正可持续使用的小工具。

## 5. 这一步完成后你学到了什么

- 会清空输入框内容
- 会主动控制光标位置
- 知道功能做通以后，还要继续优化体验

## 6. 这一步的小复盘

你刚刚做的事情是：
优化了保存后的交互体验。

它在整个项目里的作用是：
让用户可以更流畅地连续输入多条笔记。

这一步最需要记住的是：

- `.value` 可以读，也可以改
- `.focus()` 可以让输入更顺手
- 好产品不只是“能用”，还要“顺手”
